
LE MOT-CLÉ EXTENDS
Le mot clé extends en POO est utilisé pour établir une relation d'héritage entre deux classes. Lorsqu'une classe "étend" une autre classe, elle hérite de toutes les propriétés et méthodes publiques et protégées de la classe parente. Cela permet de réutiliser du code et de créer des hiérarchies de classes organisées.
